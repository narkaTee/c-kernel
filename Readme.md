# Requirements
- qemu-system-x86-64
- nasm
- xorriso
- libc6-dev-i386 (for x86 builds)

run with `make run`

TIP: To exit qemu press ALT+2 and type quit

# Error Codes
- 0 = no multiboot2 compatible bootloader
